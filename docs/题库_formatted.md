# 第一套 Java 后端开发 (1-3 年)

**考察点:** 一个是常量池中的 "xyz"，一个是堆中的 String 对象。考察内存基础

### 12. [逻辑] 两个线程同时执行 i++ (i 初始为 0) 各 100 次，最终 i 的结果可能是？

---
### 13. [JVM] 系统频繁发生 Full GC，导致系统卡顿，以下哪个原因 最不可能？
**选项:**- A. 内存泄漏，大量对象无法回收- B. System.gc () 被显式频繁调用- C. 大对象直接进入老年代，且老年代空间不足- D. 新生代设置得太大
**正确答案:** D
**考察点:** 新生代大通常会减少 Full GC 频率。考察 JVM 调优与内存模型
---
### 14. [架构] 分布式系统中，关于 "最终一致性" 的描述，正确的是？
**选项:**- A. 所有节点的数据必须时刻保持一致- B. 允许在一段时间内数据不一致，但最终会达到一致状态- C. 数据库必须使用 Serializable 隔离级别- D. 必须使用 2PC (两阶段提交) 协议
**正确答案:** B
**考察点:** 考察分布式理论 BASE/CAP
---
### 15. [Redis] 使用 Redis 实现分布式锁时，为了防止 "死锁"（持有锁的客户端挂掉导致锁不释放），必须设置？

**正确答案:** A
**考察点:** 考察分布式锁核心原理

# 前端开发 (1-3 年) 面试题

### 1. [CSS] 以下哪个 CSS 属性不属于盒子模型（Box Model）的组成部分？
**选项:**- A. Margin- B. Padding- C. Border- D. Position
**正确答案:** D
**考察点:** CSS 基础概念
---
### 2. [JS 基础] console.log (1 + '2') 和 console.log (1 - '2') 的结果分别是？
**选项:**- A. "12", -1- B. 3, -1- C. "12", NaN- D. 3, NaN
**正确答案:** A
**考察点:** JS 弱类型隐式转换：加号优先转字符串，减号优先转数字
---
### 3. [ES6] 下列关于箭头函数 => 的说法，错误的是？
**选项:**- A. 箭头函数没有自己的 this- B. 箭头函数不能作为构造函数（不能 new）- C. 箭头函数没有 arguments 对象- D. 箭头函数可以改变 this 指向（通过 call/apply）
**正确答案:** D
**考察点:** 箭头函数的 this 是固定的，call/apply 无法改变
---
### 4. [框架] React/Vue 中，组件生命周期 Mount (挂载) 完成后，通常适合做什么？
**选项:**- A. 定义 State 初始值- B. 发送网络请求获取数据- C. 销毁定时器- D. 校验 Props 类型
**正确答案:** B
**考察点:** DidMount/Mounted 是发起请求的标准时机
---
### 5. [网络] 浏览器跨域（CORS）限制主要针对的是？
**选项:**- A. CSS 文件的加载- B. Script 标签的 src 加载- C. Ajax/Fetch 网络请求- D. 图片的 src 加载
**正确答案:** C
**考察点:** 同源策略主要限制 XHR/Fetch
---
### 6. [工具] 这里的 npm install 报错，提示权限不足（EACCES），最不推荐的做法是？
**选项:**- A. 使用 sudo 强制安装- B. 检查 npm cache- C. 修改 npm 目录的归属权限- D. 使用 nvm 管理 node 版本
**正确答案:** A
**考察点:** Sudo 安装会导致后续更多权限问题，是不推荐的坏习惯
---
### 7. [逻辑] 页面白屏，控制台没有任何报错，网络请求也都 200，你首先怀疑？
**选项:**- A. 浏览器坏了- B. 这是一个 CSS 问题（如高度 0 或内容被遮挡）- C. 接口返回的数据为空，但前端没有做空状态处理- D. B 和 C 都有可能
